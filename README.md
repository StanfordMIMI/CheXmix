# CheXmix: Unified Generative Pretraining for Vision Language Models in Medical Imaging

[![arXiv](https://img.shields.io/badge/arXiv-2406.06512-b31b1b.svg?style=for-the-badge)](https://arxiv.org/abs/2604.22989)

CheXmix is a unified early-fusion generative model trained on a large corpus of chest X-rays paired with radiology reports ([CVPR Findings 2026](https://arxiv.org/abs/2604.22989)).
